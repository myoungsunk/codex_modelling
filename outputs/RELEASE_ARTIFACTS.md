# Canonical Release Artifacts

This folder contains the canonical release artifacts generated by:

```bash
make canonical_release
```

## Included files

- `report_canonical_release.md`
- `rt_dataset_canonical_release_model_params.json`
- `rt_dataset_canonical_release_synthetic_compare.json`
- `plots_canonical_release/` (P0~P21 figures)
- `rt_dataset_canonical_release.h5.part.00`
- `rt_dataset_canonical_release.h5.part.01`

## Reconstruct HDF5

GitHub file-size limits require the HDF5 to be split into parts.
Reconstruct with:

```bash
cat outputs/rt_dataset_canonical_release.h5.part.00 \
    outputs/rt_dataset_canonical_release.h5.part.01 \
    > outputs/rt_dataset_canonical_release.h5
```
